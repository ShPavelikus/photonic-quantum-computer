# Квантовые процессы в фотонных системах

## 1. Интерференция фотонов

### 1.1 Одночастичная интерференция

**Интерферометр Маха-Цендера:**
```
     ╱──────╲
|ψ⟩──BS₁──φ──BS₂──detector
     ╲──────╱
```

Вероятность детектирования:
```
P = (1/2)[1 + cos(φ)]
```

**Фазовый контроль:**
- Изменение φ влияет на интерференционную картину
- Фундаментален для квантовых вычислений

### 1.2 Двухчастичная интерференция

**Hong-Ou-Mandel (HOM) интерференция:**

Когда два неразличимых фотона падают на светоделитель 50/50 с разных входов одновременно:

```
|1⟩_A|1⟩_B ──BS──> (|2⟩_A|0⟩_B + |0⟩_A|2⟩_B)/√2
```

**Ключевые моменты:**
- Фотоны всегда выходят вместе (bunching)
- Состояние |1⟩_A|1⟩_B не возникает (деструктивная интерференция)
- Требуется идеальная неразличимость фотонов

**HOM dip:**
- График зависимости coincidence counts от временной задержки
- Минимум при нулевой задержке
- Мера неразличимости фотонов

**Условия для HOM интерференции:**
1. Одинаковая поляризация
2. Одинаковая частота (энергия)
3. Одновременное прибытие
4. Одинаковая пространственная мода

### 1.3 Многофотонная интерференция

**Boson Sampling:**
- N фотонов входят в M-модовый линейный оптический интерфейс
- Вероятность конкретного выходного паттерна:
```
P(n_out|n_in) = |Per(U_S)|²/∏m!(n_out)_m
```
где Per - перманент матрицы, U_S - подматрица унитарной матрицы

**Сложность:**
- Вычисление перманента #P-трудная задача
- Потенциальное квантовое преимущество

## 2. Квантовая запутанность

### 2.1 Bell States (Состояния Белла)

Четыре максимально запутанных двухкубитных состояния:

```
|Φ⁺⟩ = (|HH⟩ + |VV⟩)/√2 = (|00⟩ + |11⟩)/√2
|Φ⁻⟩ = (|HH⟩ - |VV⟩)/√2 = (|00⟩ - |11⟩)/√2
|Ψ⁺⟩ = (|HV⟩ + |VH⟩)/√2 = (|01⟩ + |10⟩)/√2
|Ψ⁻⟩ = (|HV⟩ - |VH⟩)/√2 = (|01⟩ - |10⟩)/√2
```

**Создание Bell states:**

Схема с CNOT и Hadamard:
```
|00⟩ ──H───●──── |Φ⁺⟩
          │
|0⟩ ──────⊕────
```

### 2.2 GHZ States (Greenberger-Horne-Zeilinger)

Многочастичное запутанное состояние:
```
|GHZ⟩_n = (|0⟩^⊗n + |1⟩^⊗n)/√2
```

Для 3 кубитов:
```
|GHZ⟩₃ = (|000⟩ + |111⟩)/√2
```

**Свойства:**
- Максимальная многочастичная запутанность
- Демонстрирует нелокальность квантовой механики
- Нарушает неравенства Мермина

**Создание GHZ:**
```
|0⟩ ──H───●───●──
          │   │
|0⟩ ──────⊕───┼──
              │
|0⟩ ──────────⊕──
```

### 2.3 W States

Альтернативное многочастичное запутанное состояние:
```
|W⟩₃ = (|100⟩ + |010⟩ + |001⟩)/√3
```

**Свойства:**
- Устойчивость к потерям: если потерян один кубит, оставшиеся все еще запутаны
- GHZ и W состояния не эквивалентны под LOCC

### 2.4 Cluster States

Используются в измерительных квантовых вычислениях (measurement-based QC):
```
|ψ⟩_cluster = ∏_{⟨i,j⟩} CZ_{ij} |+⟩^⊗n
```

**Применение:**
- One-way quantum computing
- Квантовая коррекция ошибок

## 3. Измерения

### 3.1 Проекционные измерения

**Вычислительный базис (Z-basis):**
```
M₀ = |0⟩⟨0|, M₁ = |1⟩⟨1|
P(0) = ⟨ψ|M₀|ψ⟩ = |⟨0|ψ⟩|²
```

**X-basis:**
```
M₊ = |+⟩⟨+|, M₋ = |-⟩⟨-|
|±⟩ = (|0⟩ ± |1⟩)/√2
```

**Y-basis:**
```
M_{+i} = |+i⟩⟨+i|, M_{-i} = |-i⟩⟨-i|
|±i⟩ = (|0⟩ ± i|1⟩)/√2
```

### 3.2 POVM (Positive Operator-Valued Measure)

Обобщенные измерения:
```
{E_m}, E_m ≥ 0, ∑_m E_m = I
P(m) = Tr(E_m ρ)
```

**Преимущества:**
- Более общий формализм
- Описывает измерения с потерями
- Неидеальные детекторы

### 3.3 Слабые измерения (Weak Measurements)

Измерения с малой силой связи:
- Минимальное возмущение состояния
- Получение информации о системе
- Weak values могут быть больше собственных значений

### 3.4 Измерение запутанности

**Bell measurement:**
Проекция на Bell базис:
```
{|Φ⁺⟩, |Φ⁻⟩, |Ψ⁺⟩, |Ψ⁻⟩}
```

**Схема реализации:**
```
qubit A ──●───H───M
          │
qubit B ──⊕───────M
```

## 4. Декогеренция и потери

### 4.1 Потери фотонов

**Модель потерь:**
```
ρ → (1-η)ρ_lost + η ρ
```
где η - эффективность передачи/детектирования.

**Beam splitter model:**
Потери моделируются как взаимодействие с вакуумным режимом через светоделитель.

**Влияние на вычисления:**
- Ошибки в гейтах
- Потеря информации
- Необходимость избыточности

### 4.2 Декогеренция

**Фазовая декогеренция:**
```
ρ(t) = ∑_{j,k} e^{-Γ_{jk}t} ρ_{jk}(0)|j⟩⟨k|
```

**Источники декогеренции в фотонных системах:**
- Дисперсия в оптических волокнах
- Флуктуации показателя преломления
- Термальные фотоны
- Нестабильность фазы

**Характерные времена:**
- T₁ (relaxation time): время жизни возбужденного состояния
- T₂ (dephasing time): время фазовой когерентности
- T₂ ≤ 2T₁

### 4.3 Борьба с декогеренцией

**Методы:**
1. Квантовая коррекция ошибок (QEC)
2. Decoherence-free subspaces
3. Динамическое разделение (dynamical decoupling)
4. Топологические коды

## 5. Hong-Ou-Mandel эффект (детально)

### 5.1 Теория

**Начальное состояние:**
```
|ψ_in⟩ = |1⟩_A|1⟩_B
```

**Действие светоделителя 50/50:**
```
|ψ_out⟩ = (1/2)(â†_1 + â†_2)(â†_1 - â†_2)|0⟩
        = (1/2)(â†_1² - â†_2²)|0⟩
        = (|2,0⟩ - |0,2⟩)/√2
```

**Вероятности:**
- P(1,1) = 0 (идеальный HOM dip)
- P(2,0) = P(0,2) = 1/2

### 5.2 Неидеальные случаи

**Различимые фотоны:**
```
P(1,1) = 1/2
```

**Частично различимые:**
```
P(1,1) = (1 - V²)/2
```
где V - visibility (мера неразличимости).

### 5.3 Применения

1. **Характеризация источников фотонов**
2. **Двухкубитные гейты**
3. **Квантовые схемы с post-selection**
4. **Boson sampling**

## 6. Квантовая телепортация

### 6.1 Протокол

**Цель:** Передать неизвестное состояние |ψ⟩ от Alice к Bob.

**Ресурсы:**
- Один Bell pair |Φ⁺⟩ (разделен между Alice и Bob)
- Классический канал связи

**Шаги:**

1. **Начальное состояние:**
```
|ψ⟩_A|Φ⁺⟩_{BC} = (α|0⟩ + β|1⟩)|Φ⁺⟩_{BC}
```

2. **Bell measurement Alice (кубиты A и B):**
```
Результат: один из {|Φ⁺⟩, |Φ⁻⟩, |Ψ⁺⟩, |Ψ⁻⟩}
```

3. **Классическая передача:** 2 бита информации

4. **Коррекция Bob:**
```
|Φ⁺⟩ → I (ничего)
|Φ⁻⟩ → Z
|Ψ⁺⟩ → X
|Ψ⁻⟩ → XZ
```

**Результат:** Bob получает |ψ⟩

### 6.2 Математическое описание

Полное состояние в Bell базисе Alice:
```
|ψ⟩_A|Φ⁺⟩_{BC} = (1/2)∑_i |Bell_i⟩_{AB} ⊗ U_i|ψ⟩_C
```

### 6.3 Экспериментальная реализация

**Фотонная телепортация:**
- Спонтанное параметрическое рассеяние для Bell pairs
- Поляризационные кубиты
- Детектирование совпадений

**Достижения:**
- Телепортация на > 100 км
- Квантовые повторители
- Квантовые сети

## 7. Энтропия и информация

### 7.1 Энтропия фон Неймана

```
S(ρ) = -Tr(ρ log ρ)
```

**Свойства:**
- S = 0 для чистых состояний
- S максимальна для максимально смешанного состояния

### 7.2 Mutual Information

```
I(A:B) = S(A) + S(B) - S(AB)
```

Мера корреляций между подсистемами.

### 7.3 Entanglement of Formation

Мера запутанности:
```
E(ρ) = min ∑_i p_i S(Tr_B|ψ_i⟩⟨ψ_i|)
```

### 7.4 Negativity

Мера запутанности через partial transpose:
```
N(ρ) = (||ρ^{T_B}||_1 - 1)/2
```

## 8. Практические аспекты

### 8.1 Heralding

Анонсирование создания фотона через детектирование парного фотона.

### 8.2 Post-selection

Условная успешность операций:
- Используется в линейной оптике
- Снижает эффективную скорость
- Необходим feed-forward для масштабирования

### 8.3 Multiplexing

Увеличение эффективной скорости:
- Временной multiplexing
- Пространственный multiplexing
- Частотный multiplexing
