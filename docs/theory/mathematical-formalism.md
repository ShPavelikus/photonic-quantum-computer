# Математический формализм фотонных квантовых вычислений

## 1. Представление фотонных кубитов

### 1.1 Поляризационное кодирование

Фотонные кубиты могут быть закодированы в состоянии поляризации фотона:

**Линейная поляризация (H/V кодирование):**
- Горизонтальная поляризация: |H⟩ ≡ |0⟩
- Вертикальная поляризация: |V⟩ ≡ |1⟩

**Круговая поляризация (L/R кодирование):**
- Левая круговая поляризация: |L⟩ ≡ (|H⟩ + i|V⟩)/√2
- Правая круговая поляризация: |R⟩ ≡ (|H⟩ - i|V⟩)/√2

Общее состояние поляризационного кубита:
```
|ψ⟩ = α|H⟩ + β|V⟩
```
где |α|² + |β|² = 1 (условие нормализации).

### 1.2 Пространственное кодирование (Dual-rail encoding)

В dual-rail кодировании кубит представлен наличием фотона в одной из двух пространственных мод:

- |0⟩ = |1,0⟩ (фотон в моде A, нет фотона в моде B)
- |1⟩ = |0,1⟩ (нет фотона в моде A, фотон в моде B)

Общее состояние:
```
|ψ⟩ = α|1,0⟩ + β|0,1⟩
```

**Преимущества dual-rail:**
- Естественная реализация с помощью светоделителей
- Защита от потерь фотонов (детектируем отсутствие фотона)
- Простая реализация логических операций

### 1.3 Временные бины

Кубит кодируется во временном положении фотона:
- |0⟩ - фотон в раннем временном окне
- |1⟩ - фотон в позднем временном окне

## 2. Операторы рождения и уничтожения

В квантовой оптике используются операторы второго квантования:

**Оператор уничтожения:** â
- Действует на состояние с n фотонами: â|n⟩ = √n|n-1⟩
- [â, â†] = 1 (каноническое коммутационное соотношение)

**Оператор рождения:** â†
- Создает фотон: â†|n⟩ = √(n+1)|n+1⟩
- â†|0⟩ = |1⟩ (создание одного фотона из вакуума)

**Оператор числа фотонов:**
```
n̂ = â†â
n̂|n⟩ = n|n⟩
```

### 2.1 Многомодовый случай

Для M пространственных мод:
```
â_i, â_j†, i,j = 1,...,M
[â_i, â_j†] = δ_ij
[â_i, â_j] = 0
```

## 3. Гильбертово пространство

### 3.1 Пространство Фока для N фотонов

Базис пространства Фока состоит из состояний с определенным числом фотонов:
```
|n⟩ = (â†)^n/√(n!) |0⟩
```

Для одной моды:
```
ℋ = span{|0⟩, |1⟩, |2⟩, ...}
```

### 3.2 Многокубитные системы

Для n кубитов гильбертово пространство имеет размерность 2^n:
```
ℋ_n = ℋ_1 ⊗ ℋ_2 ⊗ ... ⊗ ℋ_n
dim(ℋ_n) = 2^n
```

Базисные состояния (computational basis):
```
{|00...0⟩, |00...1⟩, ..., |11...1⟩}
```

Общее состояние n кубитов:
```
|ψ⟩ = ∑_{i=0}^{2^n-1} c_i|i⟩
```
где ∑|c_i|² = 1.

## 4. Матрица плотности

### 4.1 Чистые состояния

Для чистого состояния |ψ⟩ матрица плотности:
```
ρ = |ψ⟩⟨ψ|
```

Свойства:
- ρ† = ρ (эрмитовость)
- Tr(ρ) = 1 (нормализация)
- ρ² = ρ (идемпотентность для чистых состояний)
- Tr(ρ²) = 1 (критерий чистоты)

### 4.2 Смешанные состояния

Смешанное состояние (статистическая смесь):
```
ρ = ∑_i p_i|ψ_i⟩⟨ψ_i|
```
где p_i - классические вероятности, ∑p_i = 1.

Для смешанных состояний:
- 0 < Tr(ρ²) < 1

### 4.3 Частичный след

Для составной системы AB редуцированная матрица плотности:
```
ρ_A = Tr_B(ρ_AB)
```

Элементы:
```
(ρ_A)_ij = ∑_k ⟨i,k|ρ_AB|j,k⟩
```

## 5. Fock states и когерентные состояния

### 5.1 Fock states (Number states)

Состояния с определенным числом фотонов:
```
|n⟩ - n фотонов
n̂|n⟩ = n|n⟩
```

**Вакуумное состояние:** |0⟩
- Состояние без фотонов
- Минимальная энергия

**Однофотонное состояние:** |1⟩
```
|1⟩ = â†|0⟩
```

### 5.2 Когерентные состояния

Собственные состояния оператора уничтожения:
```
â|α⟩ = α|α⟩
```

Разложение по Fock states:
```
|α⟩ = e^(-|α|²/2) ∑_{n=0}^∞ (α^n/√(n!)) |n⟩
```

Свойства:
- Наиболее "классические" квантовые состояния света
- Минимальная неопределенность (minimum uncertainty states)
- Среднее число фотонов: ⟨n̂⟩ = |α|²

### 5.3 Сжатые состояния (Squeezed states)

Состояния с уменьшенной неопределенностью в одной квадратуре:
```
ΔX₁ΔX₂ ≥ 1/4
```

Оператор сжатия:
```
Ŝ(r) = exp[(r/2)(â²† - â²)]
```

Сжатое вакуумное состояние:
```
|r⟩ = Ŝ(r)|0⟩
```

## 6. Операции над состояниями

### 6.1 Унитарная эволюция

Квантовая эволюция описывается унитарным оператором:
```
|ψ(t)⟩ = Û(t)|ψ(0)⟩
Û†Û = ÛÛ† = Î
```

Для матрицы плотности:
```
ρ(t) = Û(t)ρ(0)Û†(t)
```

### 6.2 Измерения

Проекционное измерение в базисе {|i⟩}:
- Вероятность исхода i: P(i) = ⟨i|ρ|i⟩ = |⟨i|ψ⟩|²
- Состояние после измерения: |ψ'⟩ = |i⟩

POVM (Positive Operator-Valued Measure):
```
{M̂_i}, ∑_i M̂_i†M̂_i = Î
P(i) = Tr(M̂_i†M̂_iρ)
```

### 6.3 Тензорное произведение

Для составных систем:
```
|ψ⟩_AB = |ψ_A⟩ ⊗ |ψ_B⟩
```

В матричной форме:
```
ρ_AB = ρ_A ⊗ ρ_B
```

Размерность:
```
dim(ℋ_A ⊗ ℋ_B) = dim(ℋ_A) × dim(ℋ_B)
```

## 7. Квантовые корреляции

### 7.1 Запутанность

Состояние запутано, если его нельзя представить в виде:
```
ρ_AB ≠ ρ_A ⊗ ρ_B
```

**Bell states** (максимально запутанные состояния):
```
|Φ±⟩ = (|00⟩ ± |11⟩)/√2
|Ψ±⟩ = (|01⟩ ± |10⟩)/√2
```

### 7.2 Энтропия фон Неймана

```
S(ρ) = -Tr(ρ log₂ ρ) = -∑_i λ_i log₂ λ_i
```
где λ_i - собственные значения ρ.

Для чистого состояния: S(ρ) = 0
Максимальная энтропия: S_max = log₂(dim ℋ)

### 7.3 Schmidt decomposition

Любое чистое состояние |ψ⟩_AB можно представить как:
```
|ψ⟩_AB = ∑_i √λ_i |i⟩_A ⊗ |i⟩_B
```

Schmidt coefficients: √λ_i
Schmidt rank: число ненулевых λ_i

Для запутанных состояний Schmidt rank > 1.
